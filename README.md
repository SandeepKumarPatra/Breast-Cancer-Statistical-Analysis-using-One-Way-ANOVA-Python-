Breast Cancer Statistical Analysis using One-Way ANOVA
Project Overview

This project performs statistical analysis on a breast cancer dataset using Python.
The goal is to understand whether certain tumor features differ significantly between benign and malignant diagnoses.

The analysis manually implements One-Way ANOVA by computing:

Sum of Squares Between Groups (SSB)

Sum of Squares Within Groups (SSW)

Mean Square Between (MSB)

Mean Square Within (MSW)

F-Statistic

This helps determine whether a feature (such as mean_radius) has statistically significant differences between tumor classes.


mentation

The project manually calculates ANOVA statistics instead of using prebuilt functions.
This helps understand the mathematical foundations of statistical testing.
